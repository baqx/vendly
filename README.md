# Vendly

Vendly is an AI-powered platform that provides vendors with an **Employee as a Service**. It enables vendors to set up intelligent bots on WhatsApp and Telegram to handle product discovery, sales, payments, and logistics.

## Project Structure

- `backend/`: FastAPI implementation (Python). [Work in Progress]
- `vendly-web/`: Next.js frontend for vendors to manage their store and bots. [Existing Setup]

## Tech Stack

- **Frontend:** Next.js, TypeScript, Tailwind CSS (in `vendly-web`).
- **Backend:** Python FastAPI.
- **Integrations:** WhatsApp API, Telegram API, Interswitch (for payments), GIGL, etc.
